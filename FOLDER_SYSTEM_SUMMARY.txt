                              HOW TO USE
================================================================================

1. CREATE SAMPLE FOLDERS:
   python manage.py setup_folders

2. ACCESS DASHBOARD:
   http://localhost:8000/media/dashboard/

3. CREATE NEW FOLDER:

   - Enter name and description
   - Click "Create Folder"

   - Click folder cards to open
   - Use breadcrumbs to go back
   - Click "Root" to return to top

   - Upload media in Wagtail admin
   - Select folder from dropdown
   - Save
