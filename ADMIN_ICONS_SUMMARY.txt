================================================================================
                           VIEWSET EXAMPLE
================================================================================

class VideoViewSet(SnippetViewSet):
    model = Video
    icon = "media"                    # Icon name
    menu_label = "Videos"             # Display name
    menu_order = 203                  # Position
    add_to_admin_menu = True          # Show in menu
    list_display = [                  # List columns
        "title",
        "video_type",
        "folder",
        "duration",
        "created_at"
    ]
    search_fields = [                 # Searchable
        "title",
        "description"
    ]
    list_filter = [                   # Filterable
        "video_type",
        "folder",
        "categories"
    ]

================================================================================
                           CUSTOMIZATION
================================================================================

Change Icon:
1. Open wagtail_snippet_viewsets.py
2. Find the ViewSet class
3. Change icon = "icon-name"
4. Save and refresh

Change Menu Order:
1. Open wagtail_snippet_viewsets.py
2. Find the ViewSet class
3. Change menu_order = 200
4. Lower numbers appear higher

Add New Menu Item:
1. Create new ViewSet in wagtail_snippet_viewsets.py
2. Set icon, menu_label, menu_order
3. Set add_to_admin_menu = True
4. Register with register_snippet()

================================================================================
                           TROUBLESHOOTING
================================================================================

Icons Not Showing:
- Check icon name spelling
- Verify Wagtail version
- Clear browser cache
- Restart server

Menu Items Missing:
- Check add_to_admin_menu = True
- Verify model import
- Check for errors in console
- Restart server

Wrong Order:
- Check menu_order values
- Lower numbers = higher position
- Restart server after changes
